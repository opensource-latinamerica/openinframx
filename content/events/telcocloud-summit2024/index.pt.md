# O que você aprenderá?
- Organizações Globais:
Fundações que promovem o desenvolvimento de infraestruturas abertas.

- Computação COTS:
Os líderes na fabricação de microprocessadores, que permitem que o tráfego de telecomunicações seja movido para software em hardware genérico.

- Redes Whitebox:
Redes de alta capacidade, com fácil implantação, com a capacidade de alternar o tráfego do Core para o Acesso.

- Recursos de Rede:
VNFs e CNFs adequados para serem implantados em ambientes abertos e com sucesso em produção.

# Palestras de:
